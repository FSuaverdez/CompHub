Instructions on how to run the program

1. Download the XAMPP server.
2. In the rar file containing the source code of the project, extract it in the htdocs located in the xampp folder.
3. Run the Xampp application and start the apache and sql to launch the server.
4. Download the sql file named request.
5. Go to phpmyadmin website and create a database named request.
6. After creating a database, go to import section and import the sql file which is in the Comphub-main folder in htdocs.
7. After importing the website, you can now access the system by going into www.localhost/comphub-main/views for the user end and www.localhost/comphub-main/admin/adminIndex.php for the administrator side.
